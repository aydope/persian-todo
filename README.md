# Farmanravayi bar Zaman (Todo)

This is a simple To-Do app.
It is in Persian (RTL) and easy to use.

---

## What it can do

- Add tasks with a title and notes
- Pick a category (Personal, Work, Important)
- Mark tasks done
- Delete tasks
- Move tasks by drag & drop
- Show active tasks and today’s date

---

## Tech used

- HTML
- TailwindCSS
- jQuery
- GSAP
- Bootstrap Icons
- sessionStorage (temporary storage)

---

## How to run

Open `index.html` in a browser. No install needed.

---

## Notes

Tasks are saved only while the tab is open.

---

## Why it was made

To practice task management, drag & drop, and simple animations.
